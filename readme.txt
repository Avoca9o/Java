Мини дз по дисциплине "Конструирование программного обеспечения"
Владимиров Дмитрий, БПИ218

Тесты

Программа содержит в себе простейший класс PopcornCooker, 
умеющий делать попкорн по запросу и выкидывающий исключение
если он не умеет готовить такой попкорн

Тесты же покрывают большинство встречающихся случаев, в том 
числе и выбрасывание исключений, а также когда заканчивается
кукуруза для приготовления попкорна
